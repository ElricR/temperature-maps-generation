# temperature-maps-generation
Generation of South East France temperature maps using CVAE and cWGAN-GP architectures.

The data used in this project can be found in the following link:
https://www.data.gouv.fr/datasets/donnees-changement-climatique-sim-mensuelle

For the code to run properly, download all MENS_SIM2_[...] files as .csv.gz files.
